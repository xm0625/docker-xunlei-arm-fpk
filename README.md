# 迅雷 ARM 版 - 飞牛 NAS Docker 应用

飞牛arm版迅雷-docker版 (.fpk)

本项目仅对 cnk3x/xunlei 3.20 版本进行飞牛应用打包  
github地址： [https://github.com/cnk3x/xunlei/tree/v3.20.2](https://github.com/cnk3x/xunlei/tree/v3.20.2)  
dockerHub镜像为：cnk3x/xunlei@sha256:13a330a7280f3ff4444f47c80b4705992df3bf4d27895b6d4bc6445dfc5ffe7c  

## 功能特性

- Docker版本
- 支持飞牛ARM版
- 支持迅雷远程下载功能
- 支持手机迅雷 App 添加任务（无 3 个任务限制）

## 安装要求

- 飞牛Arm版 系统版本 >= 1.1.8

## 安装步骤

1. 下载 `.fpk` 安装包
2. 在飞牛 NAS 应用中心选择"手动安装"
3. 上传 `docker-xunlei-arm-x.x.x-arm.fpk` 文件
4. 按照安装向导配置以下参数：
   - **迅雷远程控制名称**：默认为 `fnOS`，用于在迅雷远程设备列表中的远程设备名称，比如`群晖-xxxx`，可自行修改
   - **网页访问用户名**：Web 管理界面的登录用户名。 安装好后，等待一会，点击迅雷图标，会打开迅雷账号绑定页面，输入用户名和密码，用于将当前设备绑定到你的迅雷账号的远程设备中
   - **网页访问密码**：Web 管理界面的登录密码。安装好后，等待一会，点击迅雷图标，会打开迅雷账号绑定页面，输入用户名和密码，用于将当前设备绑定到你的迅雷账号的远程设备中
   - **服务端口号**：默认为 `2345`，如无端口冲突无需修改

## 使用说明

### 访问管理界面

安装完成后，点击应用图标即可打开 Web 管理界面。

> 首次启动需要初始化，请耐心等待几分钟。

### 内测激活码

```
迅雷牛通
```

### 下载目录

默认下载路径：`文件管理` -> `应用文件` -> `docker-xunlei-arm` -> `TDDownload`

### 目录结构

| 路径 | 说明 |
|------|------|
| `TDDownload` | 下载文件存放目录 |
| `data` | 迅雷运行数据、插件、登录信息 |
| `cache` | 迅雷套件缓存 |

## 注意事项

1. **推荐使用手机迅雷添加任务**，可绕过 Web 端的3个任务数量限制
2. **下载完成后建议移动文件**：迅雷可能会对已完成的文件进行做种上传，建议将下载完成的文件移动到其他文件夹
3. **首次启动较慢**：如果安装后点击图标无法打开，请稍等片刻


### 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `XL_DASHBOARD_PORT` | Web 面板端口 | 2345 |
| `XL_DASHBOARD_USERNAME` | 登录用户名 | - |
| `XL_DASHBOARD_PASSWORD` | 登录密码 | - |
| `XL_UID` | 运行用户 ID | 1000 |
| `XL_GID` | 运行用户组 ID | 1000 |
